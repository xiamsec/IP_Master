# 🌐 NetMaster Pro

NetMaster Pro is a simple and fast subnet calculator designed for beginners and NOC engineers. It helps you quickly analyze IP addresses and CIDR values with a clean and responsive interface.

## 🚀 Features

* IP Class detection (A / B / C)
* Public vs Private IP identification
* Subnet Mask calculation
* Network & Broadcast Address
* Total IP & Usable Hosts
* First & Last Usable IP
* Responsive design (PC & Mobile)
* Clear button for quick reset

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## 📌 Usage

1. Enter an IP address (e.g. 192.168.1.10)
2. Enter CIDR (e.g. 24)
3. Click **Analyze**
4. View complete subnet details instantly

## 💡 Purpose

This tool is built for learning networking basics, preparing for interviews, and assisting in real-world NOC tasks.
